Percentual de Lance

Use para saber qual percentual o cliente vai oferecer de lance, com base no valor (R$) que ele pretende ofertar — ou o caminho inverso, partindo de um percentual desejado.

Percentual de lance—
% de Lance = Valor do lance ÷ Valor de Categoria (bem + taxas) × 100. O percentual obtido é amortizado em ordem inversa do plano (do final para o início).

Lance Embutido (Coeficiente)

No lance embutido, o cliente usa parte do próprio crédito como lance. Como incidem taxas sobre o crédito, o valor líquido recebido é menor. Informe o percentual de lance embutido desejado e o valor líquido que o cliente quer receber, para descobrir o crédito bruto necessário.

% líquido recebido—
Coeficiente—
Crédito bruto necessário—
Custo total = 100% + Taxa Adm + Fundo Reserva. % perdido = Custo total × % Lance Embutido. % líquido = 100% − % perdido. Coeficiente = 100% ÷ % líquido. Crédito bruto = Valor líquido desejado × Coeficiente.
